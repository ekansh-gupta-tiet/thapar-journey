## What I built

A glass soda bottle modelled using Revolve around a construction axis.

## Techniques used

- Construction axis (required for Revolve)
- Revolve command — profile must be a closed loop on one side of the axis
- Spline for curved bottle profile
- Sketch fully constrained before revolving

## Key rules learned

- Revolve needs a profile AND an axis — the axis must be a construction line
- The profile must be entirely on one side of the axis (no crossing)
- Spline points can be dragged but must still be constrained
