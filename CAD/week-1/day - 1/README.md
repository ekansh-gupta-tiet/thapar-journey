## What I built

A simple rectangular toy block with chamfered edges. Kevin Kennedy's first tutorial part.

## Techniques used

- New Sketch → Project/Include
- Line, Rectangle sketch tools
- Sketch constraints (horizontal, vertical, coincident)
- Extrude (one direction)
- Fillet on edges

## Key rules learned

- Every sketch must be **fully black (constrained)** before extruding
- Blue lines = unconstrained = broken model later
- Timeline at the bottom tracks every feature — never delete steps
