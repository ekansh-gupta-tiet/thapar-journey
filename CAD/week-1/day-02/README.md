## What I built

A paper clip modelled using Sweep along a 3D path.

## Techniques used

- 3D Sketch (sketch not confined to one plane)
- Sweep command — profile swept along a path
- Circular profile for the wire cross-section
- Path must be a single continuous chain

## Key rules learned

- Sweep needs two things: a profile and a path — they must be separate sketches
- 3D sketch allows lines to go in any direction (not just X/Y plane)
- The profile plane must be perpendicular to the start of the path
