## What I built

An ice cube tray with a 4×2 grid of cavities using Rectangular Pattern and Shell.

## Techniques used

- Extrude (base tray body)
- Shell command (hollow the tray)
- Rectangular Pattern (repeat cavity in X and Y directions)
- Fillet on cavity edges

## Key rules learned

- Rectangular Pattern requires a base feature first — can't pattern a sketch alone
- Shell thickness must be set before patterning internal geometry
- Pattern quantity × spacing must fit within the base body or it fails
