## What I built

A more complex bottle shape continuing from Day 2 — more detailed profile with neck geometry.

## Techniques used

- Revolve (continued from Day 2)
- More complex spline profile
- Fillet on sharp edges after revolve
- Shell command to hollow the body

## Key rules learned

- Shell after all solid geometry is done — hollowing too early removes features
- Fillet radius must be smaller than shell thickness or Fusion throws an error
- Saving as .f3d preserves the full parametric timeline
