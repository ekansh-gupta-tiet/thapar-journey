## What I built

An M8 hex nut modelled using Polygon sketch, Extrude, and Chamfer.

## Techniques used

- Polygon sketch tool (inscribed circle method)
- Extrude
- Circular cut for the thread hole (no actual thread — simplified)
- Chamfer on top and bottom faces (standard nut profile)

## Key rules learned

- Polygon in Fusion: specify number of sides + inscribed or circumscribed circle
- Chamfer angle on hex nuts is typically 30° (standard engineering practice)
- For M8: across-flats = 13mm, thickness = 6.4mm (ISO standard)

## Dimensions

- Thread size: M8
- Across flats: 13mm
- Thickness: 6.4mm
- Chamfer angle: 30°
