## What I built

A handle bar shape demonstrating when to use Loft and Sweep instead of basic Extrude.

## Techniques used

- Loft between two profiles on different planes
- Sweep along a curved path
- Guide rails on Loft for shape control
- When NOT to use Extrude (tapered/curved transitions)

## Key rules learned

- Extrude fails on shapes that taper or transition between profiles — use Loft instead
- Loft needs at least 2 profiles on different planes — more profiles = more shape control
- Guide rails in Loft control how the shape transitions between profiles
- Sweep path must be tangent-continuous (no sharp corners) for smooth results
